Ce dépôt a été créé dans le cadre d’un exercice visant à apprendre à utiliser Git et GitHub.
L’objectif principal était de découvrir :

comment créer un dépôt,
comment travailler en groupe,
comment ajouter et modifier des fichiers,
comment utiliser les branches,
comment faire des pull requests,
et comment collaborer efficacement sur une plateforme de versioning.

Chaque groupe avait pour mission de :

créer sa propre page (fichier Markdown),
y présenter son film préféré,
expliquer pourquoi ce film a été choisi,
ajouter éventuellement des images, résumés ou anecdotes,
puis intégrer cette page au projet via une pull request.

📄 Contenu du dépôt
Vous trouverez dans ce dépôt :
un dossier ou plusieurs fichiers correspondant aux différentes pages créées par les groupes ;
une présentation de chaque film préféré ;

À la fin du projet, chaque groupe a ajouté sa page au dépôt, créant ainsi une collection collaborative de présentations de films préférés, tout en apprenant les bases essentielles de GitHub.
